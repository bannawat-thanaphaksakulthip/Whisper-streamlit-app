# Whisper-streamlit-app
A simple Streamlit web app that converts speech to text using OpenAI Whisper model.

## Features
- Upload audio file (.m4a)
- Automatic speech-to-text transcription
- Language detection
- Simple UI with Streamlit

## Tech Stack
- Python
- Streamlit
- OpenAI Whisper

## Installation

```bash
git clone https://github.com/bannawat-thanaphaksakulthip/Whisper-streamlit-app
cd Whisper-streamlit-app
pip install -r requirements.txt
```

## Run App
```
streamlit run app.py
```
